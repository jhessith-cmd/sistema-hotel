# Complementación técnica aplicada

HotelControl continúa siendo el sistema principal. Se conservaron el calendario, reservas, ingresos, salidas, pagos, cancelaciones, limpieza, mantenimiento, usuarios, roles y conexión con Google Sheets.

Se añadieron del código de referencia:

- TanStack Router como contenedor de rutas.
- Página 404 en español.
- Pantalla global de recuperación ante errores.
- TanStack Query para caché y coordinación de las consultas del hotel.
- Reintento controlado de solicitudes.
- Metadatos profesionales para navegador y redes sociales.
- Idioma HTML configurado en español.
- Tipografías Fraunces y Nunito.

La interfaz y los flujos operativos existentes no fueron reemplazados.
